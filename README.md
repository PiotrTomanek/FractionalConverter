### The Utility
This app is a streamlined tool designed for construction, woodworking, and engineering. It allows you to input measurements exactly as they appear on a tape measure, bridging the gap between fractional imperial and metric systems.

### Key Features
* **Fractional Input:** Input complex measurements like 5' 8 3/4" without manual decimal conversion.
* **Integrated Math:** A built-in engine handles basic arithmetic (+, -, Ã—, Ã·) directly in the input field.
* **Seamless Toggling:** Instantly switch between **cm, mm, meters, and inches** with real-time updates.
* **OLED-Optimized Design:** High-contrast "True Black" aesthetic for maximum readability in bright environments.

### Technical Overview
Built as a lightweight web application using HTML5, CSS3, and Vanilla JavaScript. It is mobile-responsive and designed to be pinned to a home screen for a native-app experience.
